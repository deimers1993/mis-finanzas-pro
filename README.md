# Mis Finanzas Pro v3 - CORREGIDO

IMPORTANTE: Sube el CONTENIDO de esta carpeta, no la carpeta.
En Windows activa Ver > Elementos ocultos para ver .github

Pasos:
1. Crea repo nuevo VACIO en GitHub (sin README)
2. Entra en esta carpeta descomprimida
3. Selecciona TODO (Ctrl+A) - incluye .github
4. Arrastra a GitHub en Add file > Upload files
5. Commit
6. Actions > Build APK > Run workflow > descarga APK
